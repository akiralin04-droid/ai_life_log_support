# AI Life Log Support（仮）

## サイト概要
### サイトテーマ
**「日々の記録（日記）」から「企業の資産（詳細なフィードバック）」をAIが創り出す**、次世代のライフログ＆マーケティングプラットフォームです。

### 扱うコンテンツの定義：体験型コンテンツ（Experience Content）
本アプリでは、単なる商品スペックではなく、ユーザーの**「体験」**に重きを置いた以下のコンテンツをレビュー対象とします。
*   **時間消費型体験**：映画、本、ゲーム、舞台など（没入した時間に対する感想）
*   **五感体感型体験**：飲食、ガジェット、旅行先、サウナなど（身体的・感覚的な感想）

### テーマを選んだ理由
私は読書や商品購入後の「感想」を残したいと思う一方で、文章を整理する手間や、「何を書けばいいか分からない」という悩みを感じていました。
一方で、商品や作品を提供する**企業側**も、単なる評価点数だけではない、**「具体的な改善点」や「ターゲット層のリアルな反応」**を求めていますが、従来のアンケート形式では回答率が低いという課題があります。

そこで、**「AIとの対話」**と**「キャンペーン（インセンティブ）」**を組み合わせることで、課題を解決できると考えました。
ユーザーはキャンペーンに参加し、AIからのインタビューに答える形式で日記を書くだけ。AIは企業が知りたいポイントを自然に聞き出し、質の高いレビューを生成します。

「ユーザーの負担軽減」と「企業のマーケティング収集」を両立し、さらにプレゼント等の特典で「投稿の楽しさ」を提供する仕組みを実現したく、このテーマを選びました。

## 🚀 現在の開発フェーズ（MVP）
本プロジェクトはアジャイル開発の手法を取り入れ、まずは核心となる価値（Core Value）を最速で検証するため、以下の機能をMVP（Minimum Viable Product）として優先実装しています。

### MVP実装範囲（Phase 1）
1.  **AIチャット日記**：OpenAI APIを活用し、日常の記録と要約生成を行う機能。
2.  **AIレビュー生成**：日記の内容から、公開用のレビュー記事を自動生成する機能。
3.  **レビュー投稿・共有**：生成されたレビューの編集・公開機能。
4.  **ユーザー認証**：セキュアな会員登録・ログイン機能。

※ 下記「全体構想」にある企業機能やキャンペーン機能、詳細な分析機能は、フェーズ2以降の拡張実装として計画しています。

## 全体構想と実装予定の機能一覧

### 🤖 AI機能（コア機能）
OpenAI APIを活用し、ユーザーと企業の架け橋となります。
*   **AIチャット日記**：その日の出来事や感想を、AIに話しかけるように記録できます。
*   **企業連携型インタビュー（感想誘発モード）**：
    *   キャンペーン対象商品の場合、AIが「企業が聞きたいポイント（例：新作の味はどうでしたか？）」を重点的に質問します。
*   **AIレビュー生成・添削**：対話内容を元に、読みやすく整理されたレビュー記事を自動生成します。
*   **AI感情・傾向分析**：レビュー内容から感情スコアを算出し、ダッシュボードで可視化します。

### 🏢 企業・マーケティング機能（独自機能）
*   **企業アカウント**：自社の商品・作品（本、映画、グッズ等）を登録できます。
*   **キャンペーン開催機能**：「〇〇のレビュー投稿でプレゼント！」といった期間限定キャンペーンを作成できます。
*   **フィードバック分析**：自社商品に寄せられたレビューや、AIが収集した回答データを分析できます。

### 👤 ユーザー機能
*   **日記・レビュー管理**：
    *   **日記機能**：日々の記録をAIと作成できます。**公開/非公開を選択**でき、自分だけのメモとしても、フォロワーへの近況報告としても利用可能です。
    *   **レビュー機能**：日記からAIが生成した「整った記事」を公式なレビューとして投稿できます。
*   **キャンペーン参加**：トップページから開催中のキャンペーンに参加し、投稿することで特典（ランキング入賞など）を狙えます。
*   **会員登録・ログイン**：ゲストログイン（ワンクリックでお試し利用）完備。
*   **マイページ**：自分の活動履歴、感情分析グラフ、公開した日記・レビューを管理できます。

### 📝 閲覧・検索機能
*   **トップページ（ダッシュボード）**：開催中のキャンペーンや、話題のレビューを一覧表示します。
*   **検索機能**：キーワード、タグ、感情フィルター検索が可能です。

### 💬 コミュニケーション機能
*   **いいね（ブックマーク）**：参考になったレビューを評価します。
*   **コメント**：レビューや公開日記に対してリアクションを行えます。

### 🛡️ 管理者機能
*   **全体管理**：不適切なレビューや**公開された日記**の削除、悪質なユーザー/企業の利用停止処理を行います。

## ターゲットユーザ
*   **【一般ユーザー】**：日記や感想を気軽に記録したい人、お得に商品を試したい人。
*   **【企業・提供者】**：自社商品のプロモーションを行いたい、または深い顧客フィードバックを集めたい担当者。

## 主な利用シーン
1.  **ユーザー**が帰り道に、スマホのマイクに向かって「今日見た映画、ラストが衝撃的でさ…あと食べたパンケーキも美味しかった」と雑多に話しかける。
2.  **AI**がその音声を整った日記文章に変換し、「映画」と「パンケーキ」の話題が含まれていることを検知する。
3.  **AI**が「『〇〇（映画名）』と『パンケーキ』について、それぞれレビューを書きませんか？」と提案する。
4.  **ユーザー**が映画を選択すると、AIが日記の熱量を反映したレビュー記事を自動生成する。
5.  **ユーザー**がそれを微調整して公開し、同じ映画を見た人と感想を共有する。
